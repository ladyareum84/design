- em, 상위 부모의 폰트를 기준으로 한 단위 '1em = 상위 부모의 폰트 크기' 대부분의 브라우저에서 폰트 크기의 디폴트 값은 16px입니다. 1em = 16px 2em = 32px 예시 만약 시안에 있는 마진 값이 20px이라면 이건 몇 em이 될까요? 기본 폰트 크기가 16px이라고 했을 때, (20/16 = 1.25em)이라는 계산이 나오게 됩니다. - rem, 최상위 폰트를 기준으로 한 단위 '1rem = html 요소에 지정된 폰트 크기' 루트 요소의 폰트 크기를 62.5%로 지정하고 rem을 사용하면 훨씬 편해집니다. 브라우저가 지정해주는 기본 폰트 사이즈는 100% = 16px, 62.5% = 10px로 바꿔줌 아래 코드를 보면 1rem = 10px이 되어 작성 * em은 IE3부터 지원하며(어르신...), rem은 IE9부터 지원
